# Multi-agentic architecture

Multiple Agents with specialised roles working together. 

- Inputs
- Agent Commander - orchestration
- Specialised Agents - perform work
- Safeguard agent - checks and verifies work
- Collaboration and synthesis - done by the commander agent
- Output
