# A2A (Agent to Agent)

## A2A

This is an inter-agent communication protocol that enables autonomous AI Agents to discover, authenticate and use other AI agents in an asynchronous manner.

Essentially it enables Agents to find each other and collaborate & communicate to complete their work.

Key Features:

- **Asynchronous communication** (supports callbacks, polling)
- **Decentralized identity (DID)** and agent discovery
- **Supports long-running, multi-turn interactions**
- **Ideal for multi-agent systems** and distributed workflows
