# Composite UI via Microservices

Often when microservices are implemented it concentrates on the server-side elements and the frontend is a single service, i.e. a monolith.

![[Pasted image 20250401152221.png]]

This approach involves splitting the frontend down into micro services also, using composition to then piece them together into the final whole.
