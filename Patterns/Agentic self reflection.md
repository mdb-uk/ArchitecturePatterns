# Agentic self reflection

Allows agents to review their own output and refine it further. 

- Input
- Initial Output
- Reflective process
- Iteration and refinement (until Agent is satisfied with the result)
- Output

The process iterates to refine the eventual output. Multiple agents can also collaborate in this process.
