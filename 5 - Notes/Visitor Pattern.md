# Content

The Visitor design pattern is a behavioural pattern that lets you separate algorithms from the objects on which they operate.

This allows you to add new behaviour to a Visitor class, without modifying the original object. You can use a technique called Double Dispatch to then apply the correct logic based on the type without cumbersome conditionals.
