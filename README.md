Obsidian repo which details Architectural design patterns and Software Engineering patterns.

The [[Architecture Patterns Index]] file should be opened first, and from there you can browse or use the tags to locate what you want, this will open the search tab. You can use the "collapse results" option (under search settings) in order to see all files that tag appears in. 

You can hover over pattern names in the Architecture Pattern Index to get a pop up preview of that pattern file without leaving the index.

This repo is a work in progress, I'm going through and improving things gradually (including re-writing my initial rough notes and replacing pasted images with my own drawn images).
